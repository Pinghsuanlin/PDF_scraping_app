# Project Overview
It's a Python-based extraction engine designed to ingest semi-structured PDF documents (including tables, images, paragraph) , identify key tables, and normalize data into structured CSV/Excel formats for analysis.

Tools used include: LangChain, HuggingFace, Python Stremlit
